Python script for continuous speech recognition until 'q' key is pressed.

Dependencies:
- speech_recognition: Library for performing speech recognition with support for several engines and APIs.
- keyboard: Library for detecting and handling keyboard events.

Usage:
1. Run the script.
2. Select the language you want to use (1 for English, 2 for Spanish).
3. Speak into the microphone.
4. Press 'q' key to stop listening and end the program.

Features:
- Supports continuous speech recognition based on selected language.
- Handles keyboard events to stop listening when 'q' key is pressed.
- Provides error handling for speech recognition errors.

Notes:
- Ensure microphone access and internet connection for speech recognition.
- Adjust the language code ('en-US' or 'es-ES') based on your language selection.
